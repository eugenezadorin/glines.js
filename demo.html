glines.js

Simple plain-javascript tool for easy markup testing with guide lines

You don't need jQuery to use glines.js.

glines.js is mostly browser console utility, so all following methods will work in console.

To get fast API-help just type glines.help() in browser console after glines.js was included.

Download .zip or get source: http://s6-studio.ru/glines.js

Examples

Example 1

  function example1() {
    glines.addLine();
  }

Tip: You can move guide lines with mouse.

Example 2

  function example2() {
    glines.addLine('h', {color: '#f00', size: '1px'});
  }

Tip: Single click on line makes it active (focused), so you also can move it using keyboard arrows.

Example 3

  function example3() {
    glines.removeAll();
  }

Tip: If you want to remove one line, use double click on it.

How to include

If you want permanently include glines.js to your site (for example on your dev server), use standart code inside <head></head> tag:

  <html>
    <head>
      <script type="text/javascript" src="path/to/glines.js"></script>
    </head>
  </html>

If you want one-time including, there are two ways to get glines.js with browser console (type F12 to open):

jQuery way

Type this code to your browser console:

  $.getScript('http://s6-studio.ru/glines.js');

And then start using glines.js methods.

Plain-Javascript way

A bit more code:

  var s = document.createElement('script'); 
  s.src = 'http://s6-studio.ru/glines.js';
  document.body.appendChild(s);

API

There are only 3 public methods:

method description examples
glines.addLine()

Adds single guide line.

Parameters

string axis

Default value is "vertical".
Acceptable values: "vertical" or "horizontal".
You also can use short record: "v" or "h"


object customStyle

Default value is empty object. This parameter is used to rewrite default style parameters.
You can use any DOM-style attributes to change line appearance.
Additionally this object supports 2 custom properties: color and size.

Adding vertical line with default appearance:

  glines.addLine();
  glines.addLine('v'); // the same
  glines.addLine('vertical'); // the same


Adding horizontal line with default appearance:

  glines.addLine('h');
  glines.addLine('horizontal'); // the same


Try to change appearance:

  glines.addLine('v', {color: 'red', size: '10px'});
  glines.addLine('v', {backgroundColor: 'red', width: '10px'}); // the same

Size is recommended property to change line thickness instead of width for vertical lines and height for horizontal lines.
glines.removeAll()

Removes all guide lines. Method has no parameters.

Remember that you can remove each line by double click on it.
glines.help()

Use this method to get fast API-info and examples in your browser console.
